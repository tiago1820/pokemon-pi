# Pokemon Wiki - Frontend and Backend

This is a Pokemon Wiki application based on React, Redux, Node, Express and Sequelize.

## Table of Contents

- [Features](#features)
- [Demo](#demo)
- [Installation](#installation)
- [License](#license)

## Features

- Search pokemons.
- View pokemon information.
- Filter pokemons.
- Sort pokemons.
- Create new pokemon.
